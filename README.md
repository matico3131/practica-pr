# Calculadora Simple

Una calculadora básica en Python que realiza operaciones aritméticas.

## Funciones

- Suma
- Resta
- Multiplicación
- División

## Uso

```python
from calculadora import sumar, restar, multiplicar, dividir

resultado = sumar(5, 3)   # 8
resultado = restar(10, 4)  # 6
```

## Requisitos

- Python 3.8 o superior
