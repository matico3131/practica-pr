# Calculadora Simple

Una calculadora basica en Python que realiza operasiones aritmeticas.

## Funciones

- Suma
- Resta
- Multiplicacion
- Divicion

## Uso

```python
from calculadora import sumar, restar, multiplicar, dividir

resultado = sumar(5, 3)   # 8
resultado = restar(10, 4)  # 6
```

## Requisitos

- Python 3.8 o superior
